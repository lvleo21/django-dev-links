## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML, CSS e JavaScript
- Pyhone e Django
- PostgreSQL

## 💻 Projeto

O Django DevLinks é um agregador de links para usar como cartão de visitas online.
